DIRECTIONS_TO_OFFSETS

Object final static

Defined in src/util.js:48

Maps direction names to coord offsets.

Example:

`{
   up:           {x:  0, y: -1},
   up_right:     {x:  1, y: -1},
   right:        {x:  1, y:  0},
   down_right:   {x:  1, y:  1},
   down:         {x:  0, y:  1},
   down_left:    {x: -1, y:  1},
   left:         {x: -1, y:  0},
   up_left:      {x: -1, y: -1}
}`
